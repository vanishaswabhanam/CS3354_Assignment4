# CS3354 Assignment 4 - Automated Unit Test Generation

**Author:** Vanisha Swabhanam (VXS210083)

## Overview

This project compares three automated unit test generation approaches for a Java Stack implementation:
- EvoSuite - Search-based test generation
- Randoop - Random test generation
- ChatGPT - AI-assisted test generation

## Directory Structure

```
├── pom.xml
├── README.md
├── src/
│   └── Stack.java
└── test/
    ├── chatgpt/

    ├── evosuite/
    │   ├── EvoSuiteTest.java
    │   └── EvoSuiteRegressionTest.java
    └── randoop/
        ├── RegressionTest0.java
        └── ErrorTest0.java
```

## Stack Implementation

The Stack class is a fixed-size integer stack with these methods:
- `push(int value)` - Add element
- `pop()` - Remove and return top element
- `peek()` - Return top element without removing
- `isEmpty()` - Check if empty
- `isFull()` - Check if full
- `size()` - Return current size

## Test Generation

### EvoSuite Tests

Generated using:
```bash
java -jar evosuite.jar -class Stack -projectCP target/classes
```

EvoSuite uses evolutionary algorithms to maximize code coverage. It generated 16 test methods with timeout annotations and automatic assertions.

### Randoop Tests

Generated using:
```bash
java -classpath .:randoop-all-4.3.2.jar randoop.main.Main gentests --testclass=Stack
```

Randoop uses random method sequence exploration. It generated 31 regression tests and 10 error tests covering various stack operations.

### ChatGPT Tests

Generated by providing ChatGPT with the Stack.java implementation and requesting comprehensive test cases. ChatGPT produced 37 well-organized tests with clear names and comments covering all methods and edge cases.

## Running Tests

### Using Maven

```bash
mvn clean compile
mvn test
```

### Manual Compilation

```bash
javac -d target/classes src/Stack.java
javac -cp target/classes:junit-4.13.2.jar -d target/test-classes test/*/*.java
```

## Test Results

All test suites pass successfully:
- ChatGPT: 37 tests
- EvoSuite: 16 tests (+ 16 regression tests)
- Randoop: 31 regression tests + 10 error tests

## Key Findings

**EvoSuite:** Excellent automated coverage, hard to read
**Randoop:** Very fast, good at finding errors, random results
**ChatGPT:** Most readable and maintainable, similar coverage to EvoSuite
